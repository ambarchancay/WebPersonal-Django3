Definición

Un proceso es un entorno formado por todos los recursos necesarios para ejecutar programas.Para garantizar la ejecución segura de cualquier programa, los Sistemas Operativos modernos implementan un concepto: proceso.

Los Sistemas Operativos definen un grupo bastante grande de parámetros para describir el estado de parámetros o contexto de un proceso y pueden incluir el tiempo de ejecución, permisos a nivel de procesos o de usuarios, ficheros asignados,etc.

Para poder realizar esta alternancia de programas o conmutación de tareas, es imprescindible identificar y preservar el contexto de un proceso, de modo que la ejecución pueda continuar sin problemas cada vez que se retome el código de cada proceso.
